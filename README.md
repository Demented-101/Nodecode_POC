This project is for my technology enhanced learning environments, and stands as a prototype of a custom node-programming learning tool.
The final goal with this project was to create a simple node based programming language based off RecRooms own Circuits model (see here https://recroom.com/cv2)
I have made the choice to avoid using Godots built in nodes system, as making a custom system proved that I understood the requirements. 

This project is still under development!
